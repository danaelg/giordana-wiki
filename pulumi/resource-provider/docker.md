# Introduction
Pulumi peut gérer des ressources Docker via le fournisseur de ressources `Docker`

# Installation 
## Tabs {.tabset}
### Python
```bash
pip install pulumi-docker
```
> Sur un projet existant, il est préférable d'éditer le fichier `requirements.txt`
{.is-info}
